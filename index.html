Host on GitHub Pages (free, always online)

Once hosted, no laptop needs to be on. The app runs 24/7 from GitHub's servers — open the link on any phone or laptop.

  1. Go to github.com → sign in (create account if needed — it's free).
  2. Click the + icon → New repository. Name it expense-tracker.
  3. Set visibility to Private (recommended) → click Create repository.
  4. Click uploading an existing file → drag and drop your index.html → click Commit changes.
  5. Go to Settings → Pages (left sidebar) → Source: Deploy from branch → Branch: main → folder: / (root) → click Save.
  6. Wait ~2 minutes, refresh the Pages settings tab. Your URL will appear: https://yourusername.github.io/expense-tracker/
  7. Open that URL on both phone and laptop — bookmark it on the home screen of the phone.

✅ The app is now always live. You do not need your laptop on. Anyone with the link can open it, so keep the URL private.
